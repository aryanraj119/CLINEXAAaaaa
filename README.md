# CLINEXA - Complete Healthcare Platform with Doctor Dashboard

[![React](https://img.shields.io/badge/React-18-blue)](https://reactjs.org/) [![TypeScript](https://img.shields.io/badge/TypeScript-5-blue)](https://www.typescriptlang.org/) [![Tailwind CSS](https://img.shields.io/badge/TailwindCSS-3-green)](https://tailwindcss.com/)

## 🚀 Project Overview

**CLINEXA** is a comprehensive healthcare appointment booking and management platform featuring a complete **Doctor Dashboard with Patient Queue Management System**. Built with modern React, TypeScript, and Tailwind CSS, it provides end-to-end functionality for patients, doctors, and administrators.

## ✨ Key Features

### Patient Features
- 🩺 Patient registration & secure login
- 👨‍⚕️ Doctor browsing & filtering by specialization
- 📅 Appointment booking with date/time selection
- 💳 Payment integration (cash/online)
- 🩸 Blood bank management & emergency requests
- 💬 Reviews & ratings system

### Doctor Features
- 🎯 **NEW: Patient Queue Management System**
  - Real-time queue display with positions
  - Automatic wait time calculation
  - Current consultation highlighting
  - One-click start/complete/skip actions
  - Queue statistics & analytics
- 📊 Complete appointment management
- 👤 Patient information & contact details

### Admin Features
- 🗄️ Admin dashboard at `/admin/db`
- 👥 Full user & data management

## 🏥 Doctor Dashboard - Patient Queue Manager

**New flagship feature**: Real-time patient queue management with automatic wait time calculations.

```
Patient 1: 10:00 AM (Current) - Wait: 0 min
Patient 2: 10:15 AM - Wait: 15 min  
Patient 3: 10:35 AM - Wait: 35 min
```

**Actions**: Start Consultation | Mark Complete | Skip Patient
